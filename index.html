My Projects

Room Furniture Budget Calculator

This is a hackathon project I have created for the AISG Hackathon 2021. It uses python libraries PeekingDuck and TagUI RPA to help automate budget calculation for room furnitures based on themes.

Carapace AI

This is a project my friends and I have created by labeling and training datasets we have took from Kaggle to a model using Yolov5 to help in detecting whether healthcare workers have worn their personal protective equipment properly.

Carefall Vision

This hackathon project my friends and I have created for the Google for Startups Hackathon Singapore 2022. It mainly uses mediapipe for pose classification, and Google Cloud AutoML to help in predicting whether someone has fell down to determine the severity of the fall and whether it requires medical attention.
